# 📄 Kavin Kumar Portfolio — Documentation

## 📌 Overview
This project is a personal portfolio website for Kavin Kumar Subramaniam, an AI/ML engineering student.  
It showcases skills, projects, certifications, and contact details in a modern, responsive UI.

---

## 🎯 Objectives
- Present personal and academic profile
- Showcase AI/ML projects
- Highlight technical skills
- Provide contact and networking options
- Build a professional online presence
